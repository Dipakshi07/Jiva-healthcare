# Jiva HealthCare Dashboard

## Overview

Jiva HealthCare Dashboard is a React-based healthcare management application designed to manage users, family members, healthcare records, and dashboard analytics through a modern and responsive user interface.

The application provides an easy-to-use dashboard for viewing and managing healthcare-related information efficiently.

---

## Features

* User Management
* Add New User Form
* Family Member Management
* Dashboard Analytics
* Responsive UI Design
* Search and Filter Functionality
* Reusable React Components
* Modern Layout and Navigation

## Prerequisites

Before running the project, make sure the following are installed:

* Node.js
* npm (Node Package Manager)
* Git

Check installed versions:

```bash
node -v
npm -v
git --version
```

---

## Installation

### 1. Clone Repository

```bash
git clone https://github.com/your-username/jiva-healthcare-dashboard.git
```

### 2. Navigate to Project Folder

```bash
cd jiva-healthcare-dashboard
```

### 3. Install Dependencies

```bash
npm install
```

---

## Running the Application

Start the development server:

```bash
npm run dev
```

After running the command, open:

```text
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```

Generated production files will be available in the:

```text
dist/
```

folder.

---

## Preview Production Build

```bash
npm run preview
```
